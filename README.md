# Presentation-Page
Presentazione Profilo Professionale in allegato a una feedback e survey page

"Ispirandomi all'architettura dei profili Linkedin e a pagine preesistenti di questa categoria, presento un progetto (il mio primissimo) che si occupa di presentare la mia persona nel complesso, chi sono cosa mi piace fare cosa so fare, la mia formazione, le mie esperienze lavorative le mie certificazioni e contatti; il tutto si rappresenta su una singola pagina a scorrimento realizzato per ora in puro HTML per dimostrare le mie competenze attuali. Dalla seguente pagina sara possibile accedere attraverso link a una pagina di form dove sara possibile darmi una comunicazione di un certo tipo"
